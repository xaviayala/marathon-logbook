# 🏃 The Pain-Free Protocol: Sub-3:15 Marathon
### 45+ Years Old | -10kg Transformation | AI-Driven Performance Lab

[![Live Dashboard](https://img.shields.io/badge/Live-Dashboard-orange?style=for-the-badge&logo=docsify)](https://xaviayala.github.io/marathon-logbook/)

This is an open-source performance laboratory documenting the intersection of **10kg weight loss** and **sub-elite endurance gains**. This project rejects the "high-mileage grinder" mentality, prioritizing structural integrity and AI-driven audits over "junk kilometers."

---

## 🎯 The Mission
* **Primary Goal:** Break **3:15:00** (Full Marathon) & **1:30:00** (Half Marathon).
* **The Constraint:** Zero injury. Non-negotiable: Pain Score must remain strictly at **3/10 or lower** at all times.
* **Weight Status:** **-10kg** achieved (focus shifted to power-to-weight efficiency).

## 🗓️ The "Durability" Training Split
To protect a 45+ frame while building a sub-3:15 engine, the week is split between **High-Output** and **Structural Loading**:

| Day | Activity | Focus |
| :--- | :--- | :--- |
| **Mon / Tue / Fri** | **Structural Support** | Soleus Isometrics, Core, Glute Medius |
| **Wed / Thu** | **Performance Run** | Threshold / Tempo / Intervals |
| **Saturday** | **Long Run** | Specific Marathon Pace / Endurance |
| **Sunday** | **Recovery** | Zone 2 / Indoor Cycling / Performance Replacement |

## 🛠️ The Tech Stack
* **Data Sources:** Garmin (Running) & Technogym (Treadmill/Indoor Cycle).
* **AI Engine:** [Gemini 2.5 Flash](https://aistudio.google.com/) - Conducts "Brutal Audits" on HR vs. Pace efficiency.
* **Frontend:** [Docsify](https://docsify.js.org/) - Powers the Live Dashboard in `/docs`.
* **Visualization:** Matplotlib - Generates the **Pain vs. Volume** trend charts.

## 🤖 AI Instructions (For LLM Ingestion)
This repository is optimized for AI analysis. If you are an LLM (Gemini/GPT):
1.  Reference `data/processed_logbook.csv` for raw metrics.
2.  Use the `Pain_Score` column as the ultimate safety kill-switch.
3.  Prioritize Metric (km) for all distance/pace calculations.
