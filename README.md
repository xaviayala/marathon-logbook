This is an open-source "Bitacro" documenting a 10kg weight loss and elite performance 
gains without injury or "suffering." 

## 🏃 Performance Baseline
- **Half Marathon:** Sub 1:30:00
- **Full Marathon:** Sub 3:15:00
- **Status:** 45+ years old, Zero Pain.

## 🛠 The Methodology
Unlike traditional "high-mileage" grinders, this protocol prioritizes structural 
integrity over junk kilometers.

### Training Split
- **Running Days:** Wednesday or Thursday, Saturday, Sunday.
- **Structural Days:** Monday, Tuesday, Friday.
    - *Focus:* Isometría Sóleo + Core + Glúteo.

All training data is synced from **Garmin** and **Technogym**.
